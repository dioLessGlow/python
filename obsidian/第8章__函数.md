# 第8章 · 函数

**定义8.1 定义函数**
```python
def greet_user(username):
    """显示简单的问候语"""
    print(f"Hello, {username.title()}!")
```

- `def` 关键字定义函数，冒号结尾
- 文档字符串 `"""..."""` 说明函数功能
- 形参（parameter）：定义时的参数名
- 实参（argument）：调用时传入的值

**实参8.2 传递实参**
- **位置实参：**`describe_pet('hamster', 'harry')` 按顺序匹配
- **关键字实参：**`describe_pet(animal='hamster', name='harry')` 指名道姓
- **默认值：**`def describe_pet(pet_name, animal_type='dog')` 有默认值的参数放后面

**return8.3 返回值**
- `return 值` 将结果返回给调用方，函数立即结束
- 没有 return 的函数返回 `None`
- 返回字典：
```python
def build_person(first, last):
    return {'first': first, 'last': last}
```
- 让实参可选：`def build_person(first, last, age=None):`

**传列表8.4 传递列表**
- 函数内修改列表会影响到原列表（传递的是引用）
- 禁止修改：`function_name(list[:])` 传递副本

**任意参数8.5 传递任意数量的实参**
- `*toppings` 接收任意数量的位置实参（元组）
- `**user_info` 接收任意数量的关键字实参（字典）
- ```python
def make_pizza(size, *toppings):
    print(f"Making a {size}-inch pizza with {toppings}")
```

**模块8.6 将函数存储在模块中**
- `import 模块名` 导入整个模块
- `from 模块 import 函数名` 导入特定函数
- `import 模块 as 别名` 模块别名
- `from 模块 import 函数 as 别名` 函数别名
- `from 模块 import *` 导入所有函数（不推荐）

**print vs return核心区别**
`print()` 直接显示结果；`return` 交回结果，由调用方决定怎么用。函数内部有 `print()` 可直接显示，用 `return` 则外面要 `print()` 才能看到。

**return两大作用**
**1.** 返回结果   **2.** 立即结束函数。`return` 后面的代码永不执行。

**引号陷阱`return "full_name"` vs `return full_name`**
加引号返回字面量字符串，不加引号返回变量值。

**参数参数是"占位符"**
函数定义时的参数名和调用时的变量名可以不同，按位置对应即可。

**函数嵌套函数内调用函数**
```python
def a(x): print(x)
def b(y): a(y)   # 函数 b 内部调用函数 a
```

**pop移动练习8.9/8.10**
```python
def send_messages(messages, sent_messages):
    while messages:
        msg = messages.pop(0)
        sent_messages.append(msg)
```

          `pop` 实现"移动"效果：原列表减少，新列表增加。

**列表推导式小写转换**
`[user.lower() for user in users]` 创建全小写副本

**作用域局部变量**
函数内部定义的变量是局部变量，外部无法直接访问。通过参数传递。

**函数指南8.7 函数编写指南**
- 函数名用小写+下划线：`greet_user()`
- 注释用文档字符串说明功能
- 给参数指定默认值（等号两边无空格）
- 关键字实参（等号两边无空格）

**调用未定义函数定义了但没调用**
函数定义后必须显式调用才会执行，否则相当于没写。
