# 第9章 · 类

**类9.1 创建和使用类**
```python
class Dog:
    """一次模拟小狗的简单尝试"""
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def sit(self):
        print(f"{self.name} is now sitting.")
```

- `__init__()` 构造方法，创建实例时自动调用
- `self` 指向实例本身，必须作为第一个参数
- 以 self 为前缀的变量可供类中所有方法使用

**实例化根据类创建实例**
`my_dog = Dog('Willie', 6)`   访问属性：`my_dog.name`   调用方法：`my_dog.sit()`

**修改属性9.2.3 修改属性的值**
- 直接修改：`my_car.odometer_reading = 23`
- 通过方法修改：`my_car.update_odometer(23)`
- 通过方法递增：`my_car.increment_odometer(100)`

**继承9.3 继承**
```python
class ElectricCar(Car):
    def __init__(self, make, model, year):
        super().__init__(make, model, year)
        self.battery_size = 75
```

- `super()` 调用父类的方法
- 子类可以定义自己的属性和方法
- 重写父类方法：定义同名方法即可覆盖
- 可将实例用作属性：组合（composition）

**导入类9.4 导入类**
- `from car import Car` 导入单个类
- `from car import Car, ElectricCar` 导入多个类
- `import car` 导入整个模块，用 `car.Car()` 访问
- `from car import *` 导入所有类（不推荐）

**标准库9.5 Python 标准库**
`from random import randint`   `randint(1, 6)` 随机整数
`from random import choice`   `choice(list)` 随机选择

**风格9.6 类的编程风格**
- 类名驼峰命名：`Car`、`ElectricCar`
- 实例/属性/方法名小写+下划线
- 类定义后跟文档字符串
- import 语句放在文件开头

**默认值给属性指定默认值**
`self.odometer_reading = 0` 在 `__init__` 中直接赋值

**实例作属性将实例用作属性**
```python
class Battery:
    def __init__(self, size=75): ...
class ElectricCar(Car):
    def __init__(self, ...):
        self.battery = Battery()
```
