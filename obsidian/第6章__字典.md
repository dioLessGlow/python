# 第6章 · 字典

**定义6.2 使用字典**
- 字典用花括号 `{}`，键值对用冒号 `:`
- `alien = {'color': 'green', 'points': 5}`
- 访问：`alien['color']` → `'green'`
- 添加/修改：`alien['x_position'] = 0`（键不存在→添加，存在→修改）
- 删除：`del alien['points']`
- 安全取值：`alien.get('color', '默认值')`（键不存在不报错）

**遍历6.3 遍历字典**
- `for k, v in dict.items():` 键值对   **⭐ 最常用**
- `for k in dict.keys():` 或 `for k in dict:` 所有键
- `for v in dict.values():` 所有值（可能重复）
- `for v in set(dict.values()):` 所有值（去重）
- 按顺序遍历：`for k in sorted(dict.keys()):`

**嵌套6.4 嵌套**
- **字典列表：**
```python
aliens = [{'color': 'green'}, {'color': 'yellow'}]
```
- **字典中存列表：**
```python
pizza = {'crust': 'thick', 'toppings': ['mushrooms', 'cheese']}
```
- **字典中存字典：**
```python
users = {
    'aeinstein': {'first': 'albert', 'last': 'einstein'},
    'mcurie': {'first': 'marie', 'last': 'curie'}
}
```

**键名字符串键必须加引号**
`{vs1: 'value'}` 报错 `NameError`，vs1 被当作变量。正确：`{'vs1': 'value'}`

**集合vs字典区分 {} 的两种含义**
`{'a', 'b'}` 是**集合**（无序不重复）；`{'a': 'b'}` 是**字典**（键值对）

**索引字典不能用数字索引**
`dict[0]` 报错 `KeyError`。字典用键名访问：`dict['first_name']`

**调用字典不能加 () 调用**
`dict(x)` 报错 `TypeError`。用 `dict[x]` 或 `dict.get(x)`

**items()返回什么**
`dict_items([('username', 'efermi'), ('first', 'enrico')])` 键值对元组视图

**values()注意拼写**
是 `.values()` 不是 `.value()`，必须加 s

**keys()可省略**
`for k in dict` 和 `for k in dict.keys()` 效果完全一样，前者更简洁

**嵌套遍历遍历字典中的字典**
```python
for city_name, city_info in cities.items():
    for key, value in city_info.items():
        print(f"{key}: {value}")
```

**省略标记--snip--**
印刷时的省略记号，不是 Python 语法。实际代码中要替换为真实内容。

**空字典从空字典开始**
`my_dict = {}` 然后逐步添加键值对，是常见模式。

**set()去重取值**
`set(dict.values())` 获取不重复的值列表

**综合题通讯录管理系统**
编写程序 `contacts.py`，完成以下功能：

1. 创建一个字典存储通讯录，格式：`{'name': {'tel': 'xxx', 'email': 'xxx'}}`
2. 添加3个人的信息（至少包括姓名、电话、邮箱）
3. 用 `.items()` 遍历打印所有人的姓名和电话
4. 用 `.keys()` 打印所有联系人姓名
5. 用 `.values()` 打印所有邮箱地址
6. 修改其中一个人的电话，删除一个人的信息
7. 使用 `.get()` 安全查询不存在的联系人（返回默认提示）
