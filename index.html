⚙️ 基础语法与运行

▼
怎么运行 .py 文件？
在终端输入 python 文件名.py。如果已经在 Python 交互模式（>>>），先输入 exit() 退出，再运行文件。
exit()、Ctrl+D、Ctrl+Z 有什么区别？
exit() 通用；Ctrl+D 是 Linux/Mac 退出；Ctrl+Z+回车是 Windows 退出交互模式。
cd . 和 cd .. 有什么区别？
cd . 是当前目录（原地不动）；cd .. 是上级目录（返回上一层）。
PowerShell 里 cd - 报错怎么办？
PowerShell 不支持 cd -。用 pushd 记住位置，再用 popd 返回。
Python 官网是什么？
https://www.python.org，可下载安装包、查看文档。
import this 是什么？
Python 的彩蛋，输出《Python之禅》（The Zen of Python），是 Python 社区的设计哲学指南。
注释快捷键是什么？
VS Code: 中 Ctrl+/ 注释/取消注释当前行或选中的多行代码。

📝 字符串操作

▼
字符串比较时怎么忽略大小写？
用 .lower() 或 .upper() 转统一后再比较。例如：a.lower() == 'sagdgvd'
f-string 里 .title() 没生效？
错误：{name}.title() → 正确：{name.title()}，方法要放在花括号内。
removeprefix('.txt') 没去掉后缀？
removeprefix() 从开头移除，removesuffix() 从末尾移除。去后缀应该用 removesuffix('.txt')。
s.lower().title() 的结果是什么？
先全部转小写，再每个单词首字母大写。例如 "hElLo wOrLD" → "hello world" → "Hello World"。
怎么按单词切片而不是按字符？
先 split() 拆成单词列表，切片后再 join() 拼回字符串。
words = a.split()
result = ' '.join(words[0:3])
16/2 输出 8.0，怎么得到整数 8？
用整除 //：16 // 2 输出 8。或者 int(16 / 2)。
print(a) 和 a = print(p) 有什么区别？
print() 只输出到屏幕，返回值永远是 None。所以 a = print(p) 会让 a = None。

📋 列表操作

▼
append() 传两个参数报错？
append() 只能加一个元素。加多个用 extend(['ms', 'fs']) 或多次 append()。
print(ip.sort()) 为什么输出 None？
sort() 是原地排序，返回 None。正确做法：先 ip.sort()，再 print(ip)。
sort(reverse=True) 执行两次会怎样？
每次 sort() 都是重新排序。执行两次 reverse=True 会回到升序（降序→升序）。
num2 = num[:] 和 num2 = num 有什么区别？
num[:] 是复制新列表（独立）；num 是引用同一列表（一改都变）。
list(value) 报错 TypeError？
value 是整数，list() 需要可迭代对象。直接 print(value) 即可。

🔑 条件判断与字典

▼
for i in len(alien_color) 报错？
len() 返回整数，不可迭代。用 range(len(alien_color)) 生成索引序列。
if-elif 判断年龄为什么都输出"婴儿"？
if-elif 从上到下执行，满足就跳出。age >= 0 包含所有年龄，条件应从大到小排列或用区间。
空列表 users=[] 怎么处理？
空列表 for 循环体不执行。要在循环外先用 if not users: 判断处理。
用户名比较怎么不区分大小写？
创建全小写副本：current_users_lower = [u.lower() for u in current_users]，新用户名也转小写再比较。
cities={vs1:...} 报错 NameError？
字典键是字符串要加引号：'vs1':。没引号 Python 会当成变量名。
字典遍历 person[0] 报错？
字典用键访问，不是索引。用 person['first_name'] 或遍历 .items()。
.keys()、.values()、.items() 区别？
.keys() 返回键；.values() 返回值；.items() 返回键值对元组 (key, value)。
字典怎么添加键值对？
alien_0['x_position'] = 0。键不存在就添加，存在就修改。
a_0={'genn','12'} 报错？
这是集合（set），不是字典。字典要用冒号分隔键值：{'genn': '12'}。

🔄 输入与循环

▼
input() 怎么用？
返回字符串。要获取数字需转换：int(input("请输入："))。
while 循环 num+1 为什么死循环？
num + 1 只计算没赋值！要用 num = num + 1 或 num += 1。
for value in range(): print(list(value)) 报错？
value 是整数，list() 需要可迭代对象。直接 print(value) 即可。
列表输出怎么去掉引号？
引号是列表的显示格式。用 for 遍历逐个 print() 元素，不要直接 print(列表)。

🐛 代码调试

▼
print(cars==='vsvv') 报错？
Python 没有 ===，只有 ==（值相等）和 is（同一对象）。
max(num) 为什么没输出？
max() 返回结果但不自动显示。要用 print(max(num)) 才能看到。
截图里红色的 False 是什么？
VS Code: Code Runner 扩展的运行结果，不是代码，无法注释，只能关闭扩展设置。
怎么只注释部分代码？
选中要注释的行（可以部分选中多行），按 Ctrl+/ 即可批量注释/取消注释。
