基础语法与运行10

▼
运行怎么运行 .py 文件？
终端执行 python 文件名.py，要先 exit() 退出交互模式。
运行exit() / Ctrl+D / Ctrl+Z 区别？
exit() 通用；Ctrl+D 是 Linux/Mac；Ctrl+Z + 回车是 Windows。
终端cd . 和 cd .. 区别？
. 是当前目录（不动）；.. 是上级目录。
终端cd - 在 PowerShell 报错？
PowerShell 不支持 cd -，用 pushd / popd 代替。
资源Python 官网是什么？
https://www.python.org
彩蛋import this 是什么？
Python 彩蛋，输出《Python之禅》—— 20 条编程哲学。
编辑注释快捷键是什么？
Ctrl+/ 注释/取消注释当前行或选中区域。
版本怎么查看 Python 版本？
python --version 或 python -V
Gitbash 在 Windows 提示不是内部命令？
Windows 没有 bash，用 PowerShell 或 CMD。git bash 需要额外安装 Git。
GitLF will be replaced by CRLF 是什么？
Windows 换行符警告，不影响使用。git config --global core.autocrlf true 可关掉提示。

字符串操作8

▼
大小写a='Sgdgvd'，怎么比较大小写？
a.lower() == 'sgdgvd'，两边都转小写再比较。
格式化print(f"{name}.title()") 没生效？
.title() 要放在花括号内：print(f"{name.title()}")
移除removeprefix('.txt') 没去掉？
应该用 removesuffix('.txt')，从末尾移除。removeprefix 是去掉前缀。
链式s.lower().title() 结果是什么？
先全变小写，再每个单词首字母大写。如 'HELLO WORLD' → 'Hello World'
切片字符串按单词切片？
先 split() 转成列表，切片后再 join() 拼接回来。
除法16/2 输出 8.0，怎么输出整数？
用 // 整除：16 // 2 → 8，或 int(16/2) → 8
语法print(a) 和 a = print(p) 区别？
print() 只输出不返回值，返回值是 None。a = print(p) 相当于 a = None
方法字符串转首字母大写的方法？
.title() — 每个单词首字母大写；.capitalize() — 仅第一个字母大写。

列表操作7

▼
添加name.append('ms','fs') 报错？
append() 只能加一个元素。加多个用 extend([...])。
排序print(ip.sort()) 输出 None？
sort() 原地排序，返回 None。要 ip.sort() 再 print(ip)。
排序sort() 和 sort(reverse=True) 多次执行？
每次 sort() 重新排序。reverse=True 执行两次又回到升序。
复制num2 = num[:] 和 num2 = num 区别？
[:] 是复制新列表（独立）；= 是引用同一列表（一改都变）。
转换list(value) 报错？
value 是整数的话，list() 需要可迭代对象。直接用 print(value)。
遍历for i in len(alien_color) 报错？
len() 返回整数不可迭代，用 range(len(...))。
推导式列表推导式生成平方数，怎么去掉输出引号？
不要 print(整个列表)，用 for 遍历打印每个元素。

条件判断（if-elif-else）6

▼
逻辑age >= 0 导致所有年龄都是婴儿？
核心：if-elif 从上到下执行，满足一个就跳出。
age >= 0 包含了所有年龄，后面永远不会执行。
修正：条件从大到小排列或用区间： 
if age >= 65: print('老年人')
elif age >= 18: print('中青年人')
elif age >= 13: print('少年')
elif age >= 4:  print('儿童')
elif age >= 2:  print('幼儿')
else:           print('婴儿')
空列表空列表 users=[] 怎么处理？
空列表 for 循环体不执行。要在循环前判断： 
if not users:
    print('没有用户')
else:
    for user in users:
        ...
用 not users 判断列表是否为空。
大小写用户名比较不区分大小写？
创建全小写副本再比较： 
current_users_lower = [u.lower() for u in current_users]
if new_user.lower() in current_users_lower:
    print('已存在')
.lower() 只能用于字符串，不能用于列表。
错误a_0={'genn','12'} 报错？
这是集合（set）不是字典！
集合用逗号分隔元素，字典用冒号分隔键值：{'genn': '12'}
语法=== 在 Python 中报错？
Python 没有 ===，只有 ==（等于）和 !=（不等于）。
调试截图里红色 False 是什么？
VS Code Code Runner 扩展的运行结果输出，不是代码本身。

字典操作10

▼
基础字典怎么添加键值对？
字典[键] = 值：键不存在就添加，存在则修改。
访问person[0] 报错？
字典用键访问，不是索引。person['first_name'] 才对。
遍历遍历字典的三种方法？
for k in d:           # 只遍历键
for k in d.keys():   # 同上，显式写法
for v in d.values(): # 只遍历值
for k, v in d.items(): # 同时遍历键和值  ✅ 最常用
嵌套cities={vs1:...} 报错？
键是字符串要加引号：'vs1': {'xb1': '631'}
方法.items() 是什么？
返回所有键值对组成的视图对象，每个元素是 (键, 值) 元组。
配合 for k, v in dict.items(): 同时拿到键和值。
方法.keys() 和 .values() 区别？
.keys() 返回所有键；.values() 返回所有值；.items() 返回所有键值对。
错误person(xin) 语法错误？
字典不是函数，不能用 () 调用。正确：person[xin]
遍历for key in person: 和 for k,v in person.items(): 区别？
结果一样，但 .items() 更简洁、效率略高、可读性更好。
写法1：person[key] 取值；写法2：直接拿到值。
副本为什么要创建 current_users_lower 副本？
原列表 current_users 是大小写混合，不能直接 .lower()（列表无此方法）。
创建副本 [u.lower() for u in current_users]，里面每个字符串转小写，再用副本比较。
字典 vs 集合{'a', 'b'} 和 {'a': 'b'} 区别？
{'a', 'b'} 是集合（Set），无序不重复；{'a': 'b'} 是字典，键值对。

输入与循环7

▼
输入input() 怎么用？
name = input("提示文字：")  # 返回字符串
age = int(input("年龄："))  # 转整数
程序暂停，等待用户输入，按回车继续。
类型input() 返回什么类型？
字符串 str。需要数字必须手动转换：int() 或 float()
死循环while num<=5: num+1 死循环？
num + 1 只计算不保存！num 永远是 1。
正确：num = num + 1 或 num += 1
遍历for value in range(1,5): print(list(value)) 报错？
value 是整数，list() 需要可迭代对象。list(range(1,5)) 才对。
输入判断输入数字后怎么条件判断？
person = input('多少人用餐？')
if person.isdigit():        # 先判断是否数字
    if int(person) > 8:
        print('没有空桌')
    else:
        print('有空桌')
防止输入字母导致 int() 报错。
打印print('这个用户名未被使用') 没显示具体用户名？
用 f-string 带上变量：print(f"'{new_user}' 这个用户名未被使用")
换行\n 和 \t 是什么？
\n 是换行符；\t 是制表符（缩进）。

函数与调试5

▼
函数range(1, 5) 生成什么？
生成 1, 2, 3, 4（从1开始，到5结束，不含5）。
函数max() 怎么用？
max() 返回最大值，要用 print(max(num)) 才能看到输出，直接写不会显示。
语法len() 返回什么？
返回整数（长度），不能直接 for 遍历。用 range(len(list))。
调试Python 脚本运行没任何输出？
没写 print() 就不会有输出。Python 不会自动打印计算结果，必须显式 print()。
选择怎么只注释部分代码？
选中要注释的行，按 Ctrl+/（VS Code 等编辑器）。
