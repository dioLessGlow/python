# 第10章 · 文件和异常

**读取10.1 读取文件**
- `from pathlib import Path`
- `path = Path('pi_digits.txt')`
- `contents = path.read_text()` 读取全部内容为字符串
- `contents = path.read_text().rstrip()` 去除末尾空白
- **相对路径：**相对于当前运行目录   **绝对路径：**完整路径

**逐行10.1.3 访问文件中的各行**
```python
lines = contents.splitlines()
for line in lines:
    print(line)
```

**拼合10.1.4 使用文件的内容**
```python
pi_string = ''
for line in lines:
    pi_string += line.strip()
```

**写入10.2 写入文件**
- `path.write_text('内容')` 覆盖写入，文件不存在则自动创建
- 写入多行：`path.write_text('第一行\n第二行\n')`
- **追加：**`with path.open('a') as f: f.write('追加内容')`

**异常10.3 异常处理**
```python
try:
    可能出错的代码
except 异常类型:
    出错时的处理
else:
    没出错时执行
```

- `else` 块的代码只在 try 成功时执行
- 常见的异常类型：`ZeroDivisionError`、`FileNotFoundError`、`ValueError`

**静默失败10.3.8 静默失败**
```python
except FileNotFoundError:
    pass  # 什么都不做
```

          `pass` 是占位符，Python 语法要求 except 块必须有代码。

**分析文本10.3.6 分析文本**
```python
words = contents.split()
num_words = len(words)
```

**多个文件10.3.7 使用多个文件**
```python
filenames = ['a.txt', 'b.txt']
for filename in filenames:
    path = Path(filename)
    try:
        contents = path.read_text()
    except FileNotFoundError:
        pass
```

**JSON10.4 存储数据**
- `json.dumps(数据)` Python对象 → JSON字符串
- `json.loads(字符串)` JSON字符串 → Python对象
- 保存：`path.write_text(json.dumps(data))`
- 读取：`data = json.loads(path.read_text())`
- `json.dumps(data, indent=4)` 格式化输出

**重构10.4.3 重构**
将代码拆分为多个函数，各司其职：
```python
def get_stored_username(path): ...
def get_new_username(path): ...
def greet_user(): ...
```

**replace字符串替换方法**
`s = s.replace('旧', '新')` 返回新字符串，原字符串不变。

**splitlines返回列表**
`contents.splitlines()` 返回字符串列表，每行一个元素。列表没有 `.lstrip()`，需要逐行处理。

**path.exists检查文件是否存在**
`path.exists()` **必须加括号**！不加括号永远是 True。

**文件大小检查文件是否为空**
`path.stat().st_size > 0` 文件是否不为空。避免读取空文件导致 JSONDecodeError。

**文件夹创建自动创建父文件夹**
`path.parent.mkdir(parents=True, exist_ok=True)`

**保存用户数据记住用户**
```python
if path.exists() and path.stat().st_size > 0:
    username = json.loads(path.read_text())
else:
    username = input("What is your name? ")
    path.write_text(json.dumps(username))
```

**JSONDecodeError空/损坏文件处理**
```python
try:
    data = json.loads(path.read_text())
except json.JSONDecodeError:
    data = {}
```

**load vs loads区别**
`json.load(f)` 接收文件对象；`json.loads(s)` 接收字符串。
