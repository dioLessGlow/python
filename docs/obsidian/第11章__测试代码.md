# 第11章 · 测试代码

**安装11.1 使用 pip 安装 pytest**
`pip install pytest`   先 `pip install --upgrade pip` 更新 pip

**测试函数11.2 测试函数**
```python
# test_file.py
def test_full_name():
    assert get_full_name('janis', 'joplin') == 'Janis Joplin'
```

- 测试文件以 `test_` 开头
- 测试函数也以 `test_` 开头
- 使用 `assert` 断言结果是否符合预期
- 终端运行：`pytest`

**测试类11.3 测试类**
```python
def test_store_single_response():
    survey = AnonymousSurvey("What language?")
    survey.store_response('English')
    assert 'English' in survey.responses
```

**断言11.3.1 各种断言**
`assert a == b`   `assert a != b`   `assert a in list`   `assert not a in list`

**夹具11.3.4 使用夹具**
```python
@pytest.fixture
def language_survey():
    return AnonymousSurvey("What language?")
```

          夹具创建可供多个测试函数使用的资源，避免重复代码。

**未通过测试未通过时怎么办**
阅读 pytest 的错误输出，找到实际值和预期值的差异，修正代码或测试。

**添加新测试11.2.6**
每次为函数/类添加新功能时，都编写对应的测试。测试覆盖越全面，重构就越有信心。
