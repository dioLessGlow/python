# 项目1 · 外星人入侵（Pygame）

**Pygame第12章 · 武装飞船**
- `pip install pygame` 安装
- 创建游戏窗口，设置背景色，控制帧率
- 创建 Ship 类管理飞船行为和图像
- 响应键盘事件，控制飞船移动
- 添加子弹：Bullet 类，限制子弹数量

**外星人第13章 · 外星人**
- 创建 Alien 类，生成外星舰队
- 外星舰队移动：向右→下移→向左循环
- 子弹与外星人碰撞检测
- 飞船与外星人碰撞 → game over
- 外星人到达屏幕底部 → game over

**记分第14章 · 记分**
- 创建 Play 按钮
- 提高难度：每轮加速
- 记分牌：显示得分、最高分、等级、剩余飞船数

**整体结构游戏循环与事件驱动**
- `_check_events()` 处理输入
- `_update_screen()` 更新画面
- 主循环：事件 → 更新 → 绘制，循环往复

**综合题扩展外星人入侵游戏**
在第12-14章的外星人入侵项目基础上，增加以下功能：

1. 添加一种"Boss 外星人"，生命值是普通外星人的 5 倍
2. Boss 每隔一段时间发射子弹，玩家需要躲避
3. 增加道具系统：击落外星人有概率掉落加速/护盾/三连发
4. 添加音效：开枪、爆炸、道具拾取
5. 显示游戏时长和命中率统计
6. 将最高分保存到 JSON 文件中，跨游戏持久化
