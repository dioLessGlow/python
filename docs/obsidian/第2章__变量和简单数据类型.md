# 第2章 · 变量和简单数据类型

**变量2.2 变量**
- 变量命名规则：字母/下划线开头 + 字母/数字/下划线，区分大小写
- 不能使用 Python 关键字和内置函数名
- 变量是标签（引用），不是容器

**字符串2.3 字符串方法**
- `name.title()` 首字母大写   `name.upper()` 全大写   `name.lower()` 全小写
- **f-string：**`f"Hello, {name}!"` 在字符串中嵌入变量
- `\n` 换行符   `\t` 制表符（缩进）
- `rstrip()` 去右空白   `lstrip()` 去左空白   `strip()` 去两侧
- `removesuffix('.txt')` 删除后缀   `removeprefix('http://')` 删除前缀

**字符串f-string 中调用方法**
`f"{name.title()}"` — 方法调用放在花括号**内部**

**字符串链式调用**
`s.lower().title()` 先全变小写再首字母大写

**数字2.4 数字类型**
- `+ - * /` 加减乘除（除法结果始终为浮点数）
- `//` 整除   `%` 取余   `**` 乘方
- 任意两个数相除结果都是浮点数：`16/2 → 8.0`
- 数中的下划线：`1_000_000` 等价于 `1000000`（提高可读性）
- 同时给多个变量赋值：`x, y, z = 1, 2, 3`
- 常量：全大写命名 `MAX_CONNECTIONS = 5000`（约定，不强制）

**注释2.5 注释**
`# 这是注释`   注释说明代码"为什么这样做"而非"做了什么"

**类型转换str() / int() / float()**
`str(123) → "123"`   `int("123") → 123`   `float("3.14") → 3.14`

**f-string输出带引号的字符串**
`f'"{city},{altion}"'` 外层单引号包内层双引号，双引号成为普通字符

**printprint() 的返回值**
`print()` 只输出不返回值，`a = print("x")` 时 `a` 为 `None`

**彩蛋2.6 Python 之禅**
终端运行 `import this` 显示 20 条 Python 编程哲学，核心：*Beautiful is better than ugly.*

**快捷键VS Code 注释快捷键**
`Ctrl+/` 注释或取消注释当前行/选中区域

**字符串替换replace()**
`s = s.replace('旧', '新')` 返回新字符串，原字符串不变（不可变特性）

**切片字符串按单词切片**
先 `split()` 转列表，切片后再 `join()` 拼回字符串

**inputinput() 返回字符串**
`input()` 永远返回 `str`，需要数字时手动转换：`int(input(...))`

**错误常见的 NameError**
变量名拼写错误、未定义就使用、字符串忘了加引号 都会引发 `NameError`
