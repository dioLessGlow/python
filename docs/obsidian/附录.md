**Git附录D · 使用 Git 进行版本控制**
- `git init` 初始化仓库
- `git add .` 暂存   `git commit -m "描述"` 提交
- `git status` 查看状态   `git log` 查看历史
- `git remote add origin URL` 连接远程
- `git push origin main` 推送
- `git branch -M main` 重命名分支
