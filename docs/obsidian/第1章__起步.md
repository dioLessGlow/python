# 第1章 · 起步

**编程环境1.1 编程环境简介**
- Python 版本：建议使用最新版（本书用 Python 3.x）
- 运行代码片段：在终端输入 `python` 进入交互式环境，`exit()` 退出
- 编辑器：VS Code（免费、跨平台、功能强大）

**环境搭建1.2 搭建编程环境**
- **Windows：**从 python.org 下载安装程序，勾选"Add Python to PATH"
- **macOS：**从 python.org 下载安装包
- **Linux：**系统通常已预装，终端运行 `python3 --version` 检查

**Hello World1.3 运行 Hello World 程序**
- VS Code 安装 Python 扩展
- 创建 `hello_world.py`，写入 `print("Hello World!")`
- 终端运行：`python hello_world.py`

**终端操作1.5 从终端运行 Python**
- Windows 用 `python 文件名.py`
- macOS/Linux 用 `python3 文件名.py`
- `cd 目录名` 切换目录；`cd ..` 返回上级；`dir`(Windows)/`ls`(Mac/Linux) 查看文件列表

**问题排除1.4 排除安装问题**
- 无法识别 `python` 命令：检查是否添加到 PATH
- Windows 可以用 `py` 代替 `python`
- 推荐使用文本编辑器（VS Code）而非 IDLE

**小结本章目标**
搭建好 Python 编程环境，成功运行第一个 Python 程序。

**综合题搭建你的第一个 Python 项目**
编写程序 `my_info.py`，完成以下功能：

1. 在终端打印 `Hello, Python!`
2. 创建变量 `name` 存储你的名字，`age` 存储你的年龄，并用 f-string 输出 `我叫XX，今年XX岁`
3. 使用 `input()` 让用户输入喜欢的编程语言，输出 `你也喜欢XX啊！`
4. 给代码添加注释说明每步的作用
