# 第4章 · 操作列表

**for循环4.1 遍历整个列表**
```python
for item in list:
    print(item)
```
注意冒号 `:` 和缩进。`for` 循环只是"查看"，不会修改原列表。

**缩进4.2 避免缩进错误**
- 忘记缩进 → `IndentationError`
- 不必要的缩进 → 逻辑错误
- 循环后不必要的缩进 → 循环结束还执行
- 遗漏冒号 `:` → 语法错误

**range()4.3 创建数值列表**
- `range(1, 6)` → 生成 1~5（含头不含尾）
- `list(range(1, 6))` → `[1, 2, 3, 4, 5]`
- `range(1, 10, 2)` → 步长为2：`[1, 3, 5, 7, 9]`
- `min(list)` `max(list)` `sum(list)` 统计计算

**切片4.4 使用列表的一部分**
- `list[0:3]` 前三个元素
- `list[:4]` 从头到索引4之前
- `list[2:]` 从索引2到最后
- `list[-3:]` 最后三个元素
- 遍历切片：`for item in list[:3]:`

**元组4.5 元组**
- 不可变列表，用圆括号 `()`：`dimensions = (200, 50)`
- 可遍历但不能修改元素
- 不能修改元组元素，但可以给元组变量重新赋值

**代码格式4.6 设置代码格式**
- 缩进：每级4个空格（不要混用Tab和空格）
- 行长：建议不超过79字符
- 空行：用空行分隔代码块，提高可读性
- PEP 8 是 Python 代码风格指南

**遍历`for i in len(list)` 报错**
`len()` 返回整数不可迭代。用 `for i in range(len(list)):`

**复制`[:]` 创建列表副本**
`list_copy = original_list[:]` 创建独立副本，修改副本不影响原列表。

**统计列表统计计算**
`min(digits)` `max(digits)` `sum(digits)`
