# 第5章 · if语句

**条件测试5.2 条件测试**
- `==` 检查相等（区分大小写）
- `.lower()` 忽略大小写比较
- `!=` 检查不等
- ` =` 数值比较
- `and` / `or` 检查多个条件
- `in` / `not in` 检查值是否在列表中
- 布尔值：`True` / `False`

**if语句5.3 if 语句结构**
```python
if 条件:
    执行
elif 条件:
    执行
else:
    执行
```

- 从上到下依次判断，满足一个就跳出（不继续往下）
- 可省略 `else`，使用 `elif` 更明确
- 测试多个条件要用多个 `if`（不用 elif）

**空列表5.4.2 确定列表非空**
`if list:` 非空为 True；`if not list:` 空为 True

**多列表5.4.3 使用多个列表**
```python
for user in new_users:
    if user.lower() in [u.lower() for u in current_users]:
        print("已存在")
```

**区间判断推荐写法**
```python
if age >= 65: print("老年人")
elif age >= 18: print("中青年人")
elif age >= 13: print("少年")
else: print("儿童/婴儿")
```

          条件从大到小排列，避免被第一个条件覆盖。

**区间判断使用范围**
`if 0 <= age < 2:` Python 支持链式比较

**等于Python 没有 ===**
只有 `==`（值相等）和 `!=`（不等），没有 `===`

**成员检查值是否在列表中**
`if 'admin' in users:` / `if 'admin' not in users:`

**布尔布尔表达式**
布尔值就是 `True` 或 `False`，常用于跟踪状态

**忽略大小写.lower() 用于字符串**
`.lower()` 是字符串方法，列表没有！要逐个元素转换：`[u.lower() for u in list]`

**逻辑错误条件覆盖问题**
`if age >= 0:` 包含了所有年龄，后面的 `elif` 永远不会执行。条件必须精心设计顺序。

**格式5.5 设置 if 语句的格式**
在比较运算符两边各加一个空格：`if age >= 18:` 而不是 `if age>=18:`

**综合题电影票价格计算器**
编写程序 `ticket.py`，完成以下功能：

3. 使用 `and` 实现区间判断的另一种写法
