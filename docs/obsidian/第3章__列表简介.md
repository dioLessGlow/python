# 第3章 · 列表简介

**定义3.1 列表是什么**
- 列表用方括号 `[]` 表示，元素用逗号分隔：`bicycles = ['trek', 'cannondale', 'redline']`
- 访问元素：`bicycles[0]` 第一个，`bicycles[-1]` 最后一个
- 索引从 **0** 开始，不是 1

**修改3.2.1 修改元素**
`motorcycles[0] = 'ducati'`

**添加3.2.2 添加元素**
`list.append('新元素')` 末尾添加   `list.insert(索引, '新元素')` 指定位置插入

**删除3.2.3 删除元素**
- `del list[索引]` 删除指定位置
- `list.pop()` 弹出末尾   `list.pop(索引)` 弹出指定位置
- `list.remove('值')` 按值删除第一个匹配项

**排序3.3 管理列表**
- `list.sort()` 永久排序（改变原列表）
- `list.sort(reverse=True)` 永久反向排序
- `sorted(list)` 临时排序（不改变原列表）
- `list.reverse()` 反转列表顺序
- `len(list)` 获取列表长度

**复制复制列表**
`new_list = old_list[:]` 创建独立副本；`new_list = old_list` 只是引用（一改都变）

**追加多个append vs extend**
`append()` 加一个元素；`extend([a, b, c])` 加多个元素

**排序注意sort() 返回 None**
`list.sort()` 原地排序返回 `None`，应先排序再打印列表

**索引3.4 避免索引错误**
索引从 0 开始，最后一个是 `-1`。访问不存在的索引会报 `IndexError`

**推导式列表推导式**
`[x**2 for x in range(1, 6)]` → 生成 `[1, 4, 9, 16, 25]`。遍历打印每个元素不会显示引号。
