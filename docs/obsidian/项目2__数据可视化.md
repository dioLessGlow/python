# 项目2 · 数据可视化

**Matplotlib第15章 · 生成数据**
- `pip install matplotlib`
- 折线图：`plt.plot(x, y)`
- 散点图：`plt.scatter(x, y)`
- 颜色映射（cmap）、自动保存 `plt.savefig()`
- 随机漫步：RandomWalk 类
- Plotly：`pip install plotly`，交互式图表

**CSV第16章 · 下载数据**
- 解析 CSV 文件头提取数据
- datetime 模块处理日期
- 绘制温度图、给区域着色
- GeoJSON：全球地震散点图

**API第17章 · 使用 API**
- `pip install requests`
- 调用 GitHub API 获取仓库数据
- Plotly 可视化 + 自定义工具提示 + 可点击链接
- Hacker News API

**Plotly交互式图表**
Plotly 能生成鼠标悬停显示数据的交互式图表，适合 Web 展示。

**综合题天气数据可视化仪表盘**
使用 Matplotlib 和 API 制作天气仪表盘：

1. 使用 requests 调用 OpenWeatherMap API 获取你所在城市的7天天气预报
2. 用 Matplotlib 绘制温度折线图（最高温/最低温双线）
3. 用散点图显示每天的湿度分布
4. 用 Plotly 制作交互式图表，鼠标悬停显示详细数据
5. 将图表保存为 PNG 文件
6. 使用随机漫步生成模拟天气变化轨迹
